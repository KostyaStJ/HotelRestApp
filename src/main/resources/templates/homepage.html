Introduction

This is simple REST Application for hotel booking service.

Guide

  • Create hotel rooms
  • Create user
  • Book room for a few days

Working with users

  1. Create new user

    2. http://localhost:8080/users/add

    Params Example
    login ?login=test
    name &name=John
    lastName &lastName=Doe

  2. Show all users

    3. http://localhost:8080/users/all

Working with rooms

  1. Add new room

    2. http://localhost:8080/rooms/add

    Params Example Available values
    number ?number=1
    type &type=Lux Econom, Standart, Lux, PresidentLux
    price &price=200

  2. Get rooms

    • All

      • http://localhost:8080/rooms/all

    • By room category

      • http://localhost:8080/rooms/category

      Params Example Available values
      category ?type=Econom Econom, Standart, Lux, PresidentLux

    • By price

      • http://localhost:8080/rooms/price

      Params Example
      price ?price=200

    • By dates

      • http://localhost:8080/rooms/available

      Params Example
      startDate ?startDate=2018-11-12
      endDate ?endDate=2018-11-15

Working with bookings

  1. Book room

    2. http://localhost:8080/bookings/book

    Params Example Available values
    startDate ?startDate=2018-11-12
    endDate ?endDate=2018-11-15
    roomNumber ?roomNumber=5
    login ?login=user1
    options ?options=UseSafe, Minibar CleaningRoom, Safe, Telephone, Minibar, DryCleaning;

  2. Get all bookings

    3. http://localhost:8080/bookings/all

  3. Get user bookings

    4. http://localhost:8080/bookings/user

    Params Example
    login ?login=user1
